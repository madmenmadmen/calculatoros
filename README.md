# CalculatorOS

**Bare Metal Calculator** — операционная система-калькулятор, работающая напрямую на железе (без Linux/Windows).  
Поддерживает BIOS и UEFI (через CSM).

## 📦 Возможности

### ➗ Арифметика
- Сложение (`+`), вычитание (`-`), умножение (`*`), деление (`/`), остаток (`%`)
- Степень (`^`)
- Квадратный корень (`sqrt`)
- Скобки и унарный минус

### 📊 Функции
| Команда | Описание | Пример |
|---------|----------|--------|
| `rch x n` | Округление до n знаков | `rch 3.14159 2` → `3.14` |
| `ceil x` | Округление вверх | `ceil 3.14` → `4.00` |
| `floor x` | Округление вниз | `floor 3.99` → `3.00` |
| `abs x` | Абсолютное значение | `abs -5` → `5.00` |
| `pct x total` | Процент от числа | `pct 20 200` → `10.00%` |
| `vat x [rate]` | НДС (по умолч. 20%) | `vat 100` → `20.00` |
| `sum a b c ...` | Сумма чисел | `sum 10 20 30` → `60.00` |
| `avg a b c ...` | Среднее арифметическое | `avg 10 20 30` → `20.00` |
| `min a b c ...` | Минимум | `min 5 2 8` → `2.00` |
| `max a b c ...` | Максимум | `max 5 2 8` → `8.00` |

### 🔧 Переменные
- `var name = value` — создать переменную (int/double/string)
- `echo name` — вывести значение
- `type name` — показать тип
- `list_vars` — список всех переменных
- `delete name` — удалить переменную

### 🎨 Интерфейс
- `bg-color <color>` — цвет фона (black, blue, red, green, yellow и др.)
- `char-color <color>` — цвет текста
- `cursor-color <color>` — цвет курсора
- `clear` / `cls` — очистить экран
- `exit` — перезагрузка
- `time` — показать текущее время (из CMOS)

### ⌨️ Редактор строки
- Стрелки **←** **→** — навигация по тексту
- **Backspace** — удаление символа

## 🛠️ Сборка и запуск

### Зависимости
- `i686-elf-gcc`
- `i686-elf-ld`
- `grub-mkrescue`
- `qemu-system-x86_64` (для тестирования)
